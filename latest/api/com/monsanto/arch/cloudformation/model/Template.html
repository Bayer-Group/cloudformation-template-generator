Template

Related Docs: object Template | package model

case class Template(Description: String, Parameters: Option[Seq[Parameter]], Conditions: Option[Seq[Condition]], Mappings: Option[Seq[Mapping[_]]], Resources: Option[Seq[Resource[_]]], Outputs: Option[Seq[Output[_]]], AWSTemplateFormatVersion: String = "2010-09-09") extends Product with Serializable

Template is the container for all the elements of your AWS CloudFormation template.

// create the template
val simpleTemplate = Template(
  AWSTemplateFormatVersion = "2010-09-09",
  Description = "Simple S3 Bucket Template",
  Resources = Some(
    Seq(
      `AWS::S3::Bucket`(
        name = "S3Bucket",
        BucketName = Some("UniqueBucketForSimpleTemplate")
      )
    )
  ),
  Parameters = None,
  Conditions = None,
  Mappings = None,
  Outputs = None
)
